# Fullstack Node.js and React.js Ecommercial Web Application

## Installation

### Backend
``` 
$ git clone https://github.com/nmtoan251998/cacao-app.git

$ cd cacao-app

$ npm install
```
### Frontend
In **/cacao-app**
```
$ git clone https://github.com/nmtoan251998/cacao-app-view.git

$ cd cacao-app-view

$ npm install
```
